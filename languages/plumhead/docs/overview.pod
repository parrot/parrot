=head1 NAME
 
Plumhead - Introduction

=head1 What is Plumhead?

An incomplete implementation of PHP on Parrot.
Plumead is short for Plum-headed Parakeet.

=head1 Goals

Show an example for using XSLT for creating PAST.

=head1 Implementation

Take XML output from PHC. XSLT creates PIR that sets up PAST.

=head Dependencies

xsltproc and ghc need to be in PATH.

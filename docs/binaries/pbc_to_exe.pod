=head1 Name

pbc_to_exe.pod - Generate executables from Parrot bytecode

=head1 Description

Compile bytecode to executable.

=head1 Usage

pbc_to_exe [option] <file>

=head2 Comman line Options

For help use :

    -h
    --help

To insall use :

    -i
    --insall

To runcore or to change runcore use :

    -R
    --runcore=slow|fast

For output file use :

    -o
    --output=FILE

To change garbage collectro algorith use :

    --gc=ms2|gms

C<gms>  is default. C<ms2> is older and slower

=head2 Synopsis

    pbc_to_exe my.pbc

Will generate : 

    my.exe

And 
 
    pbc_to_exe my.pbc --install

Will generate :

    installable_my.exe

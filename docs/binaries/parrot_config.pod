=head1 Name

parrot_config.pod - Prints a Parrot Configuration item.

=head1 VERSION

version 0.01

=head1 DESCRIPTION

Print out configuration items.

=head1 SYNOPSIS

    ./parrot_config --dump 

With this it will print every available key. 
With specific key it will print only it's corresponding value.

    ./parrot_config VERSION 

This for example print you the version of parrot.

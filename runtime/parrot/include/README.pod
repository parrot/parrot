=over 4

=item include files with constants defined somewhere in the interpreter
(I<see> F<config/gen/parrot_include.pl> for details)

=item configuration data (config.fpmc)

=item bytecode used at runtime (parrotlib.pbc)

=back

Additionally there are some snippets of PIR code:

=over 4

=item hllmacros.pir provides some nifty PIR macros

=item test_more.pir provides boilerplate code for test scripts written in PIR
